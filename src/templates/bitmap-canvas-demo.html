The main thread creates a ImageBitmapRenderingContext with const ctx = canvasBitmap.getContext("bitmaprenderer");, the web worker creates a OffscreenCanvas, render something into it, converts the canvas into a bitmap with const bitmap = canvas.transferToImageBitmap(); and sends the bitmap to the main thread as a Transferable. The main thread updates its canvas with ctx.transferFromImageBitmap(event.data.payload.bitmap);
